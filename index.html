Мой стэк технологий

Я учусь как самостоятельно, так и с помощью менторов. По книгам, статьям, обучающим видео.

HTML

Язык, который создавался для того, чтобы помочь браузеру понять из каких частей состоит веб-страница, и какую смысловую нагрузку эти элементы несут.

SASS

Препроцессор своего рода расширение, созданное для упрощения каскадных таблиц стилей CSS.

Java Script

Язык программирования, который добавляет интерактивность на ваш веб-сайт.

Bootstrap

Фреймворк, который используется для быстрого создания адаптивных дизайнов сайтов.

jQuery

Библиотека JS, позволяющая быстро и просто добавлять улучшения в дизайн и поведение веб-страниц.

MySQL

Реляционная система управления базами данных.

Github

Веб-сервис основанный на системе контроля версий Git.

Gulp

Таск-менеджер для автоматического выполнения часто используемых задач.

Wordpress

Cистема управления контентом на сайте.
